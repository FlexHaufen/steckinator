This repository contains the software for the **Motor Controller** powered
by a Raspberry Pi Pico

---

## 🛠️ Setup

1. Clone repository
2. Import with [Raspberry Pi Pico Extention](https://marketplace.visualstudio.com/items?itemName=raspberry-pi.raspberry-pi-pico) in VScode

---

## 🚀 Deploy
1. Compile Project
2. Connect Raspberry Pi Pico over USB with host PC
3. Enter `BOOTSEL` mode
    - Press and hold the `RST` button
    - Press and hold the `BOOTSEL` button
    - Release the `RST` button
    - Release the `BOOTSEL` button
4. Press `Run Project (USB)`
